Ray Tracing in Unity

April 2025

This was my first physics-based Unity mini project, inspired by Sebastian Lague. It simulates a light ray reflecting off surfaces in 3D space using real-time raycasting and vector reflection.

What It Does

  • Fires a light ray from an origin point in a direction
  • Detects collision using Physics.Raycast
  • Reflects the ray off surfaces using Vector3.Reflect()
  • Visualizes all of this in the Unity scene view

Core Concepts

  • Raycasting
  • Surface normals
  • Vector math for reflections
